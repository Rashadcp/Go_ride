# 🚕 Go Ride – Hybrid Ride Sharing Platform

Go Ride is a **production-level ride-sharing platform** that combines:

* 🚕 Taxi Booking (Instant Ride)
* 🚗 Carpool (User-driven rides)
* 💰 Wallet (Razorpay Integration)
* 📡 Real-time Tracking (WebSockets)
* 🚨 Emergency Safety System
* ⭐ Rating & Feedback System

---

## 🧠 CORE FLOW

```text
Search → Select Ride Type → Ride → Track → Pay → Rate → Complete
```

---

## 🎯 KEY FEATURES

* Real-time ride system (Socket.IO)
* Hybrid ride model (Taxi + Carpool)
* Wallet + Razorpay integration
* Live map tracking
* Emergency reporting system
* Rating & feedback system
